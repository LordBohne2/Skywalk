# Skywalk

A simple Garry's Mod addon that lets you jump in the air.

This is my first lua addon for garry's mod.
It may be a bit junky but it works (at least for me).
Feel free to post the error messages so i can fix them when i have time.

## About Skywalk
It is an addon where you can jump in the air. you can jump normally forward, to the side or backward. You can also jump up or down quickly. If it is too slow for you, you can also jump forward very fast.

## Controlls:
- Space: Jump into the sky.
- Key(W): Jump in the direction you are looking.
- Keys(A, S, D): Jump in the direction with a normal air jump.
- Shift: Jump fast
- CTRL: Cancel Movement

## Settings
- Enable or Disable it
- Can Change Base Speed of Skywalk
- Can Change Base Jump Height
- Enable or Disable Sound
- Enable or Disable Particle

## Features
- Air Jump
- Jump in the Direction you are watching (Press W)
- A S D Air Jump
- Speed Jump (Press Shift)
- Stop Movement in the Air (Press CTRL)
- Jump Fast Up (Watch in the Sky)
- Jump Fast Down (Watch on the Ground)
- Sound File
- Settings
  - Enable/Disable Skywalk
  - Enable/Disable Sound
  - Enable/Disable Particle
  - Change Base Speed
